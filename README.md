# Hycon java utilities

Common java functions for Hycon.



### Install dependencies

```bash
<dependency>
  <groupId>com.hycon</groupId>
  <artifactId>Hyconj-Util</artifactId>
  <version>0.0.3</version>
  <type>pom</type>
</dependency>
```
